Project User Stories:

1.	As a user, I want to sort comments by proximity to me.

PRIORITY: 1

TEST CASES: 

User tests:
	A. Top comment sorting by proximity to the user:
		1. In the top comment list view activity, select sort by proximity.
		2. Confirm that the top comment list is sorted by proximity closest to the user in descending order.
	B. Reply comment sorting by proximity to the user:
		1. In the top comment list, select a top comment to bring up the reply comment list view activity.
		2. In the reply comment list view activity, select sort by proximity.
		3. Confirm that the reply comment list is sorted by proximity closest to the user in descending order.

10.	As a user, I want comments that I read or comments that I've indicated I want to read, to be locally cached so I can read them when I am not on the internet.

PRIORITY: 2

TEST CASES:

User tests:
	A. Caching read comments:
		1. In the top comment list, select a top comment to bring up the reply comment list view activity.
		2. Back out of the reply comment list.
		3. Disconnect from the internet.
		4. Select the same top comment from step 1.
		5. Confirm that this brings up the reply comment list view activity with all the reply comments.
	B. Caching comments set to read later:
		1. In the top comment list, set a top comment to read later.
		2. Disconnect form the internet.
		3. Select the same top comment from step 1.
		4. Confirm that this brings up the reply comment list view activity with all the reply comments.
